## 硬件组成
* 计算机的硬件系统由五大部分组成：运算器、控制器、存储器、输入设备（如鼠标键盘）、输出设备（如显示器）
* 存储器分为内部存储器（即内存，容量小、速度快、临时存放数据）和外部存储器（即硬盘、光盘等，容量大、速度慢，长期保存数据）。
* 输入设备和输出设备合并称为外部设备，即外设。
* 主机： CPU + 主存储器
## 中央处理单元CPU
* 中央处理单元（CPU）组成：由运算器、控制器、寄存器和内部总线组成。
* 中央处理单元功能：实现程序控制、操作控制、时间控制、数据处理功能。
* 运算器组成：由算数逻辑单元 ALU（实现对数据的算数和逻辑运算）、累加寄存器AC()
