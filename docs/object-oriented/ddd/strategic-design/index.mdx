### 业务背景（面向自动售卖机的零售SaaS）
![image-20240108223617526](https://wkq-img.oss-cn-chengdu.aliyuncs.com/img/image-20240108223617526.png)

* 商品中台中能获取所有的目前能售卖的商品数据，并且提供检索接口。
* 配送中台

### 团队背景

* A 公司：SaaS 公司，主要面向零售企业客户。
* SmartRM 产品团队：一位资深产品经理，几位产品策划。
* SmartRM 研发团队：一位架构师，10+开发，资深开发 3 ~ 4 位。
* B 集团：零售行业某头部商家。
* 全国各地有大量商超和便利店，有成熟的内部系统和供应链。
* 基于自动售卖机的零售业务是新业务，希望借助 SmartRM 来完成新业务的搭建。

### 案例优势

* 复杂度可覆盖 DDD 大部分知识点并体现其价值
* 智慧零售市场庞大且场景接近日常生活
* 有数据分析需求，可结合大数据分析场景
